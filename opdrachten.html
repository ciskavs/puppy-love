Opdrachten Bootstrap CoderDojo


De programmeur heeft de website Puppy Love gemaakt. Kijk maar hier. Maar het ziet er nog niet erg mooi uit, vind je ook niet? Aan jou de taak om de website er beter uit te laten zien! Dat ga je doen met Bootstrap.


Opdracht 1 - bouw het ontwerp na


Stap 1: Code downloaden

De code die je straks gaat aanpassen kun je vinden op Github. Klik op de groene knop 'Clone or Download' en download het zip bestand. Pak het daarna uit.


Stap 2: Project openen in editor

Open de editor van jouw keuze (bv. Sublime of Atom of een andere editor) en open daarin het project dat je net gedownload hebt. Als het goed is zie je een map 'css' en een map 'images' en een aantal bestanden, waaronder index.html.


Stap 3: Bootstrap gebruiken

Om Bootstrap te kunnen gebruiken moet je het normaal gesproken eerst downloaden en zorgen dat de browser weet dat je het wil gebruiken. Maar nu hebben we dat al voor je gedaan: in het mapje css vind je het Bootstrap bestand: bootstrap.min.css. Dit bestand is klein gemaakt door onder andere de spaties te verwijderen. Het is daardoor niet zo goed leesbaar. Maar alle informatie over Bootstrap kun je vinden op de Bootstrap website.
In het bestand index.html vind je deze regel: <link rel="stylesheet" href="css/bootstrap.min.css">. Hiermee is aangegeven dat de browser Bootstrap moet gebruiken.


Stap 4: Aan de slag!

  • In het bestand ontwerp.pdf kun je het voorbeeld zien van hoe de website eruit moet komen te zien.
  • De code die je aan moet passen staat in index.html. Je hoeft zelf geen HTML te schrijven en ook geen CSS.
  • Met Bootstrap kun je niet de hele website opmaken. Om de grootte, kleur en type van de letters goed te zetten gebruik je het css bestand puppy-styles.css. Je kunt de namen van de classes op dezelfde manier gebruiken als je classes van Bootstrap gebruikt.

Tip: de meeste programmeurs maken de website stap voor stap op. Eerst zetten ze alles op de goede plaats (door het gebruik van het Grid (Rooster) uit Bootstrap). Daarna gaan ze de details invullen, zoals de kleur en grootte van de letters en de afstanden tussen plaatjes en tekst.


Opdracht 2 - Responsive maken


Nu de website er op een groot scherm goed uitziet, gaan we de website 'responsive' maken. Responsive betekent dat de website er goed uitziet op verschillende apparaten, zoals op desktop computerschermen, tablets en mobiele telefoons. Open wat je in opdracht 1 hebt gemaakt en maak je scherm maar eens kleiner. Ziet dat er goed uit? Als je nog wel wat verbeterpuntjes ziet, dan is deze opdracht voor jou!

Voor deze opdracht heb je het grid systeem (rooster) van Bootstrap nodig.


Stap 1: mobiele telefoons

In de vorige opdracht heb je voor 'Mijn favoriete puppies' steeds drie puppies naast elkaar laten zien. Maar op een mobiele telefoon passen er geen drie plaatjes van puppies naast elkaar. Dan moet je naar rechts scrollen om alle plaatjes te kunnen zien. Dat willlen we natuurlijk niet! Op mobiele telefoons willen we ze liever onder elkaar zien. Kun je dat maken?

Tip: gebruik voor de kleinste schermen (< 768px) 'xs', bijvoorbeeld col-xs-12.


Stap 2: tablets

Tablets zijn kleiner dan desktop schermen en net als de mobiele telefoon passen er geen drie plaatjes naast elkaar zonder dat je hoeft te scrollen naar rechts. Maar als de plaatjes onder elkaar staan, is er steeds veel witruimte naast de plaatjes. Ook niet zo mooi. Kun jij het zo maken dat er op tablets steeds twee plaatjes naast elkaar staan?

Tip: gebruik voor tablets (768-960 px) 'md', bijvoorbeeld col-md-6.


Opdracht 3


De website Puppy Love ziet er nu helemaal uit zoals in het voorbeeld èn ziet er goed uit op verschillende apparaten. Maar je hebt vast wel leuke ideeën om 'm nóg mooier te maken. Maak er met behulp van Bootstrap wat moois van!

Om je wat ideeën te geven:

  • Maak van elke favoriete puppy een kaartje. Het kaartje heeft een gekleurde rand en in het kaartje staat het plaatje van de puppy, zijn naam, de informatie tekst en een link naar Wikipedia. Hiervoor heeft Bootstrap card. (Je moet zelf ook wat HTML schrijven)
  • Maak een slideshow van puppy plaatjes. Hiervoor heeft Bootstrap carousel. ( Je moet zelf ook wat HTML schrijven)
