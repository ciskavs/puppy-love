Opdrachten Bootstrap CoderDojo



Opdracht 1

De programmeur heeft de basis voor de website Puppy Love gemaakt. Kijk maar in het bestand index.html. Maar het ziet er nog niet erg mooi uit, vind je ook niet? Aan jou de taak om de website er beter uit te laten zien! Dat ga je doen met Bootstrap.
Om Bootstrap te kunnen gebruiken moet je het normaal gesproken eerst downloaden en zorgen dat de browser weet dat je het wil gebruiken. Maar nu hebben we dat al voor je gedaan: in het mapje css vind je het Bootstrap bestand: bootstrap.min.css. Dit bestand is klein gemaakt door onder andere de spaties te verwijderen. Het is daardoor niet zo goed leesbaar. Maar alle informatie over Bootstrap kun je vinden op de Bootstrap website.
In het bestand index.html vind je deze regel: <link rel="stylesheet" href="css/bootstrap.min.css">. Hiermee is aangegeven dat de browser Bootstrap moet gebruiken.

Aan de slag!

  • In het bestand 'Ontwerp website puppy love.pdf' kun je het voorbeeld zien van hoe de website eruit moet komen te zien.
  • De code die je aan moet passen staat in index.html. Je hoeft zelf geen HTML te schrijven en ook geen CSS.
  • Met Bootstrap kun je niet de hele website opmaken. Om de grootte, kleur en type van de letters goed te zetten gebruik je het css bestand puppy-styles.css. Je kunt de namen van de classes op dezelfde manier gebruiken als je classes van Bootstrap gebruikt.


Opdracht 2

Nu de website er op een groot scherm goed uitziet, gaan we de website 'responsive' maken. Responsive betekent dat de website er goed uitziet op verschillende apparaten, zoals op destop computerschermen, tablets en mobiele telefoons. Maak je scherm maar eens kleiner. Ziet dat er goed uit? Als je nog wel wat verbeterpuntjes ziet, dan is deze opdracht voor jou!
In de vorige opdracht heb je voor 'Mijn favoriete puppies' steeds drie honden naast elkaar laten zien. Maar op kleinere schermen (768-960 px) willen we dat dat er nog maar twee zijn. En op nog kleinere schermen (< 768px) moeten de honden onder elkaar komen, zodat je op een mobiele telefoon niet naar rechts hoeft te scrollen om alle honden te kunnen zien. Voor deze opdracht heb je het grid systeem van Bootstrap nodig.
